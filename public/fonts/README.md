## Fallbacks:
If T26 Carbon files are not available, the site will fall back to:
1. JetBrains Mono (for mono class)
2. Inter (for sans class)
3. System fonts

## Usage:
The font is automatically applied to:
- All body text (`font-sans`)
- All mono text (`font-mono` and `.mono` class)
- New `font-carbon` class for explicit T26 Carbon usage

The CarbonBold.ttf provides the distinctive T26 Carbon look throughout all games!
